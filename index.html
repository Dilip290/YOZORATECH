About the Job

This is a full-time, on-site role for a Web Application Developer located in Hyderabad. The Web Application Developer will be responsible for developing and maintaining custom web applications, designing back-end systems, and implementing scalable solutions to meet client requirements. The role includes collaborating with cross-functional teams, writing and testing clean, efficient code, and optimizing applications for performance and scalability. The developer will contribute to system architecture design and ensure alignment with project goals and deadlines.

Company Description

Yozora Technologies, a Hyderabad-based startup founded in 2024, specializes in creating scalable and high-performance web solutions tailored to drive business growth and user engagement. With expertise in custom web applications, e-commerce platforms, UI/UX design, and API development, Yozora Technologies emphasizes scalable architecture, user-centric design, and performance optimization. Combining modern technology, agile methodologies, and a commitment to quality assurance, the company ensures timely delivery of innovative web solutions. Yozora Technologies takes pride in offering responsive and client-focused services to help businesses establish exceptional digital presences.

Qualifications

  • Proficiency in Web Application Development and Back-End Web Development
  • Experience in building and maintaining Web Applications and Software Development
  • Strong understanding of Object-Oriented Programming (OOP)
  • Ability to collaborate in a team environment and meet delivery timelines
  • Knowledge of modern frameworks, version control systems, and responsive design principles
  • Bachelor's degree in Computer Science, Software Engineering, or a related field
  • Experience with APIs, database management, or agile development processes is a plus

Key Responsibilities

  • Develop and maintain custom web applications
  • Design and implement back-end systems
  • Write and test clean, efficient code
  • Optimize applications for performance and scalability
  • Collaborate with cross-functional teams
  • Contribute to system architecture design
  • Ensure alignment with project goals and deadlines
